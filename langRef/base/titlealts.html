<titlealts>

The <titlealts> element contains alternate titles, such as a title intended for use in a table of contents or a title specific to search results.

Attributes

The following attributes are available on this element: Universal attribute group.

Example

The following code sample shows how the <titlealts> element contains a search title:

<topic id="programming-example">
  <title>Example of required programming</title>
  <titlealts>
    <navtitle>Example of programming in XSLT</navtitle>
  </titlealts>
  <body>
    <!-- ... -->
  </body>
</topic>
