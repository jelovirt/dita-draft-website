<featnum>

The <featnum> element contains the feature number of a product in the metadata.

Attributes

The following attributes are available on this element: Universal attribute group.

Example

<prodinfo>
 <prodname>BatCom</prodname>
 <vrmlist>
  <vrm version="v5r2"/>
 </vrmlist>
 <featnum>135</featnum>
 <component>TCP/IP</component>
</prodinfo>
