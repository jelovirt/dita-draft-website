<topicCell>

The <topicCell> element contains topics that will be associated with subjects in each following column of the current row in the <topicSubjectTable>.

Specialization hierarchy

The <topicCell> element is specialized from relcell. It is defined in the classification-domain module.

Attributes

The following attributes are available on this element: Universal attribute group and Attributes common to many map elements. This element also uses @scope, @format, and @type from Link-relationship attribute group.
