<prolog>

The prolog contains metadata about the topic, for example, author information or subject category.

Attributes

The following attributes are available on this element: Universal attribute group.

The following attributes are available on this element: Localization attributes, Metadata attributes, and @class.

Example

The following code sample shows a <prolog> element that contains common metadata items:

<prolog>
  <author>Paul Norman</author>
  <copyright>
    <copyryear year="1930"/>
    <copyrholder>Paul Norman</copyrholder>
  </copyright>
</prolog>
