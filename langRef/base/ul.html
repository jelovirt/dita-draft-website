<ul>

An unordered list is a list in which the order of items is not significant.

Attributes

The following attributes are available on this element: Universal attribute group, compact, and spectitle (specialization attributes).

The following attributes are available on this element: Universal attribute group.

Example

The following code sample shows a list in which the order of items is unimportant: 

<p>Here are the countries that I have visited:</p>
<ul>
 <li>Germany</li>
 <li>France</li>
 <li>Japan</li>
 <li>Mexico</li>
</ul>
