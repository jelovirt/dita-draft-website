<navtitle>

A navigation title is an alternative title for a resource. It is designed for situations where the topic title is unsuitable for use in a table of contents or navigation pane.

Usage information

The <navtitle> element is a convenience element. It is equivalent to a <titlealt> element with @title-role set to navigation.

Processing expectations

Processing expectations are dictated by the rules for the <titlealt> element.

In some cases, when processing a <topicref> element that has no @href attribute, the navigation title can also be used as the title of the generated topic, if applicable.

Specialization hierarchy

The <navtitle> element is specialized from <titlealt>. It is defined in the alternative-titles domain module.

Attributes

The following attributes are available on this element: universal attributes and @title-role.

For this element, @title-role has a default value of navigation.
