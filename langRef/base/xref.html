<xref>

A cross reference is an inline link. A cross reference can link to a different location within the current topic, another topic, a specific location in another topic, or an external resource such as a PDF or Web page.

Attributes

The following attributes are available on this element: universal attributes, link-relationship attributes, and @keyref.

The following attributes are available on this element: link-relationship attributes, localization attributes, @class, @keyref, and @outputclass.
