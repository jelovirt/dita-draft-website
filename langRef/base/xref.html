<xref>

A cross reference is an inline link. A cross reference can link to a different location within the current topic, another topic, a specific location in another topic, or an external resource such as a PDF or Web page.

Attributes

The following attributes are available on this element: Universal attribute group, Link-relationship attribute group, and @keyref.

The following attributes are available on this element: Link-relationship attribute group, Localization attribute group, @class, @keyref, and @outputclass.
