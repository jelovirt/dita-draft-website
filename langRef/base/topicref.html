<topicref>

A topic reference is the mechanism for referencing a topic (or another resource) from a DITA map. It can nest, which enables the expression of navigation and table-of-content hierarchies, as well as containment hierarchies and parent-child relationships.

Attributes

The following attributes are available on this element: Universal attribute group, Link-relationship attribute group (with a narrowed definition of @href, given below), Attributes common to many map elements, Topicref-element attributes group, @keys, and @keyref.

The following attributes are available on this element: Attributes common to many map elements (with a narrowed definition of @locktitle given below), ID attribute group, Link-relationship attribute group (with a narrowed definition of @href, given below), Localization attribute group, Metadata attribute group, @class, @keyref, @keys, and @outputclass.

@href
Points to the resource that is represented by the <topicref>. See The href attribute for detailed information on supported values and processing implications. References to DITA content cannot be below the topic level: that is, you cannot reference individual elements inside a topic. References to content other than DITA topics should use the @format attribute to identify the kind of resource being referenced.
@locktitle
The value of the @locktitle attribute is set to yes.
